# 🤖 Agent Modes

**Custom mode export files for the RooForge orchestration pipeline**

</div>

---

## Overview

These YAML files define the 5 custom modes for Roo Code. Import them via **Settings → Custom Modes → Import**.

## Modes

| Mode | File | Description |
|------|------|-------------|
| 🎯 Orchestrator | [`orchestrator-export.yaml`](orchestrator-export.yaml) | Strategic planning, logical task bounding, and project oversight |
| ⚡ Subtask Orchestrator | [`subtask-orchestrator-export.yaml`](subtask-orchestrator-export.yaml) | Atomic subtask decomposition and high-precision execution management |
| 🏗️ Architect | [`architect-export.yaml`](architect-export.yaml) | Plan, design, think, and strategize before implementation |
| 🔍 Ask | [`ask-export.yaml`](ask-export.yaml) | Get answers and explanations from the Web or the Codebase |
| 🔀 Git | [`git-export.yaml`](git-export.yaml) | Conventional commits, branch management, and repository integrity |

## Installation

Download the YAML files from the [latest release](../../releases/latest) and import each one into Roo Code via **Settings → Custom Modes → Import**.
